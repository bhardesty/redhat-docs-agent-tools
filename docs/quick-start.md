# Quick start

1. Add the marketplace:

    ```
    /plugin marketplace add https://github.com/redhat-documentation/redhat-docs-agent-tools.git
    ```

2. Install a plugin:

    ```
    /plugin install hello-world@redhat-docs-agent-tools
    ```

3. Use a command:

    ```
    /hello-world:greet
    ```
